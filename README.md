# 🚀 ESP Connection App

A lightweight Android application designed to connect ESP-based devices (e.g., ESP32 / ESP8266) to Wi-Fi networks using **EspTouch (SmartConfig)**.

This app serves as a **core component of the E-ComposThink IoT System**, enabling seamless device provisioning and communication between hardware and the mobile application.

---

## 🔗 Role in E-ComposThink System

This application is responsible for connecting ESP-based sensors to Wi-Fi, allowing them to transmit data to the system.

It enables communication between:

* 📡 ESP hardware (sensors)
* 📱 Mobile application
* ☁️ Backend/database system

Without this setup, real-time monitoring and data collection will not function.

---

## 📱 Features

* 📡 Connect ESP devices to Wi-Fi using SmartConfig (EspTouch)
* ⚡ Fast and simple setup process
* 📲 Mobile-first interface (Android APK)
* 🔗 Designed for integration with IoT-based systems
* 🧩 Works as a companion app for the Capstone project

## 🛠️ How It Works

1. Open the app
2. Enter your Wi-Fi credentials (SSID & Password)
3. Start the connection process
4. The app sends the credentials to the ESP device via SmartConfig
5. ESP device connects to the Wi-Fi automatically

---

## 🔄 How It Fits in the System

1. Install ESP Connection App
2. Connect ESP device to Wi-Fi using SmartConfig
3. ESP device starts sending sensor data
4. Data is received and displayed in the main mobile application dashboard

## 🧠 Technology Used

* **Flutter** (Frontend)
* **EspTouch / SmartConfig Protocol**
* **ESP32 / ESP8266 compatibility**

---

## ⚠️ Notes

* Ensure the ESP device is in **SmartConfig mode** before connecting
* Both phone and ESP device should be within the same network range
* Works best on **2.4 GHz Wi-Fi networks**
